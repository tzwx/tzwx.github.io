sci论文写作基础
Created 2021-02-26| Updated 2021-02-26|sci写作
Word count:615|Reading time: 2 min
| Post View:|comments:

原则:

  • 写作要站在 读者 的角度来写，而不是作者的角度，以 reviewer 的角度来看；
  • 论文不能锋芒太强，容易有负面印象；
  • 每个很小的东西要考虑读者懂不懂，即使前面已经提过也要考虑是否再次提示；
  • 故事要讲完整，且故事前后要串起来，逻辑不能有漏洞；
  • 实验越充分越好

一、命名规范：

论文建好一个latex工程，先按照此模板建立目录结构，命名规范便于引用、修改

  • 图片 命名：、
  • 新建 Figures 文件夹，图片放在文件夹下；
  • 图片以 Fig1-PTM.jpg 方式明命，即 序号+描述
  • 文件 命名：
  • cvpr.tex 会议/期刊名称.tex
  • cvpr_supplementary.tex 补充材料 会议/期刊名称_supplementary.tex
  • References.bib 参考文献 bib 文件命名

二、写作格式规范

写作要规范统一，任何一个小问题都很致命，例如图片引用、空格问题等

  • 空格问题
  • 每两个单词之间、两句话间要有一个空格，不多不少
  • \cite{} 前后有空格
  • 加粗问题
  • 粗体表示强调，粗体不能随意使用；注意统一问题
  • 简写格式
  • 举例说明 e.g. 后面要跟 逗号，空格，用斜体包裹
  • 等等 etc. 后面小点不能省
  • 解释说明 i.e., 后面的逗号，前后空格，用斜体包裹
Code
  • 图片表格的引用
  • 图的引用 Fig.~\\ref{fig:results} Fig. 后面要加空格 Fig. 1,
  • 表的引用 Table 1 Table加空格，不加点， 后面跟引用
